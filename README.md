# Action-Rougelite-Prototype
Top-down action roguelite prototype made in Godot. Focuses on procedural dungeon generation, basic combat systems, enemy AI, and gameplay architecture experiments.

## Planned Features
- procedural dungeon generation
- top-down movement
- enemy AI (basic chase)
- combat system
- simple game loop (death → restart)
- item/loot system
- basic progression system
